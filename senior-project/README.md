# Senior Project — 3-DOF Mobile Manipulator

> 🚧 **Work in progress** — code and CAD files will be added here.

## 📌 Overview

This project is a **3-DOF robotic arm** mounted on a mobile base, developed as a senior capstone project. It builds on the experience gained from the [semester project (5-DOF KUKA youBot)](../semester-project/README.md) and focuses on a custom-designed arm with its own mechanical and software stack.

---

## 📦 Project Structure

| Folder | Description |
|---|---|
| `matlab/` | Control, kinematics, and dynamics scripts |
| `Coppeliasim_Scenes/` | `.ttt` scene files for 3D simulation (if applicable) |
| `cad/` | CAD design files for the custom 3-DOF arm |
| `csv files/` | Motion and control data for analysis and replay |
| `docs/` | Project report and documentation |
| `media/` | Images and videos for documentation |

---

## ⚙️ Key Differences from Semester Project

| Feature | Semester Project (5-DOF KUKA) | Senior Project (3-DOF Custom) |
|---|---|---|
| Robot | KUKA youBot | Custom-designed |
| DOF | 5 | 3 |
| Base | Mecanum-wheel omnidirectional | Mobile base |
| CAD source | CATIA V5 → URDF | Custom CAD files |
| Simulator | CoppeliaSim + ROS + MATLAB | MATLAB + CoppeliaSim |
